Mentsruly <br>
This is a webapp named Menstruly repository in which the early detection of PCOS is made using DNN model which is a deep learning model which tells the user that whether she has pcos or not
using the prediction model by filling the form , the form is need to be filled by the clinical data of the user. <br>
It also tracks her menstrual cycles and also mood patterns and then she can also gets a visual representaion of her cycle and mood swing patterns.<br>
It also includes a community page in which now can add only text posts , this phase helps the user to connect with other users and also to share her
experiences and suggestions with other users also.<br>
